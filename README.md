# AvatarAdjuster
Расширка, чтобы двигать и менять масштаб картинок внутри аватарки

## Установка

1. Откройте SillyTavern → **Extensions** → **Install extension**
2. Вставьте ссылку:

```text
https://github.com/mooxsy72-oss/AvatarAdjuster
```

3. Нажмите **Установить** — расширение установится автоматически

4. Перезагрузите страницу, найдите в верхнем левом углу от аватарки шестеренку
<img width="311" height="208" alt="image" src="https://github.com/user-attachments/assets/9bc86751-3732-43a0-8c6a-60f3e93fdf26" />

5. меняйте размер и двигайте как угодно
